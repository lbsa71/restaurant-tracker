# 11-star-rating

> Epic: Reviews & Ratings
> Dependencies: 09-restaurant-detail

## Goal

Reusable star rating component for display and input.

## Requirements
- StarRating component with 1-5 stars
- Display mode: shows filled/empty stars based on value
- Input mode: clickable stars to select rating
- Half-star support for averages (display only)
- Unit tests for component

## Done when
- [ ] `npm run build` passes
- [ ] `npm test` passes with StarRating tests
- [ ] Component works in both modes
