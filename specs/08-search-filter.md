# 08-search-filter

> Epic: Search & Filter
> Dependencies: 07-restaurant-detail

Lägg till sök och filter på restauranglistan.

## Krav
- Sökfält för namn
- Filter: cuisine, rating, price_range, favorites
- Sortering: name, rating, visited_at
- URL-params för filter state

## E2E Test
Skriv `e2e/search-filter.spec.ts` som verifierar:
- Sök filtrerar listan
- Filter fungerar
