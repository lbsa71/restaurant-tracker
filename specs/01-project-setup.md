# 01-project-setup

> Epic: Foundation
> Dependencies: None

## Goal

Set up Vite + React + TypeScript + Tailwind with dark mode polestar-inspired design.

## Requirements
- Vite with React 18 and TypeScript strict mode
- Tailwind CSS with dark mode as default
- Design tokens: dark bg (#0a0a0a), light text, minimal spacing
- Clean typography (Inter or similar)

## Done when
- [ ] `npm run dev` starts dev server
- [ ] `npm run build` passes
- [ ] Dark mode styling visible
- [ ] Folder structure matches CLAUDE.md
