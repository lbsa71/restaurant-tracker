# 03-supabase-setup

> Epic: Database & Auth
> Dependencies: 02-testing-setup

## Goal

Initialize Supabase local development with config and client.

## Requirements
- `supabase init` run
- supabase/config.toml configured
- src/lib/supabase.ts client file
- .env.local.example with required vars

## Done when
- [ ] `supabase start` works
- [ ] Supabase client exports correctly
- [ ] .env.local.example documents VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY
