# 14-search-bar

> Epic: Search & Discovery
> Dependencies: 08-restaurant-list

## Goal

Search bar component to find restaurants by name or cuisine.

## Requirements
- SearchBar component with text input
- Debounced search (300ms delay)
- Search by name, city, or cuisine_type (case insensitive)
- Updates restaurant list with results
- Clear button to reset search

## Done when
- [ ] `npm run build` passes
- [ ] Typing in search filters restaurant list
- [ ] Clear resets to all restaurants
