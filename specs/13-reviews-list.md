# 13-reviews-list

> Epic: Reviews & Ratings
> Dependencies: 12-review-form

## Goal

Display list of reviews on restaurant detail page.

## Requirements
- ReviewsList component showing all reviews for restaurant
- Each review shows: user display_name, rating, review_text, visited_at
- Sorted by created_at descending (newest first)
- useReviews hook for fetching

## Done when
- [ ] `npm run build` passes
- [ ] Reviews display on restaurant detail
- [ ] Shows reviewer name and rating
