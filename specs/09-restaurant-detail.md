# 09-restaurant-detail

> Epic: Restaurant CRUD
> Dependencies: 08-restaurant-list

## Goal

Restaurant detail page with all info and booking link.

## Requirements
- /restaurants/:id page
- Show: name, address, city, cuisine, category
- "Book a table" button linking to booking_url (opens new tab)
- Show average rating and review count
- Reviews section (list of reviews)

## Done when
- [ ] `npm run build` passes
- [ ] Detail page shows restaurant info
- [ ] Booking link opens in new tab
