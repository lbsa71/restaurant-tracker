# 08-restaurant-list

> Epic: Restaurant CRUD
> Dependencies: 07-add-restaurant

## Goal

Display list of all restaurants with cards.

## Requirements
- / (home) shows restaurant list
- RestaurantCard component: name, city, cuisine, average rating
- useRestaurants hook fetching from Supabase
- Empty state when no restaurants
- Link to restaurant detail page

## Done when
- [ ] `npm run build` passes
- [ ] Restaurant cards display on home page
- [ ] Click card navigates to detail
