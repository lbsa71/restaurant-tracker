# 17-profile-page

> Epic: User Profiles
> Dependencies: 13-reviews-list

## Goal

User profile page showing user info.

## Requirements
- /profile page (own profile, requires auth)
- /users/:id page (view any user's public profile)
- Show: display_name, avatar, member since date
- Edit profile button on own profile
- Stats: total reviews, restaurants added
