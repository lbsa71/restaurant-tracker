# 07-restaurant-detail

> Epic: Core Restaurant CRUD
> Dependencies: 06-restaurant-form

Skapa detaljsida för enskild restaurang.

## Krav
- Visa all info om restaurangen
- Rating som stjärnor
- Price range som $-symboler
- Map-länk för address
- Edit och Delete knappar
- Bekräfta innan delete

## E2E Test
Skriv `e2e/restaurant-detail.spec.ts` som verifierar:
- Detalj-sida laddar
- Delete-bekräftelse visas

## Klart när
- [ ] `npm run build` passerar
- [ ] E2E-test passerar
- [ ] Delete fungerar med bekräftelse
